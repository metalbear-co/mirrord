Added `feature.db_branches[].connection_settings` for PostgreSQL database branches. The map of settings (GUCs) is applied to every source connection mirrord opens while building the branch via `PGOPTIONS`, so it is in effect for the schema dump and data copies. The common use is a Row-Level Security tenant variable that the copy's `SELECT` must read, but any session GUC (such as `role` or `search_path`) works.
