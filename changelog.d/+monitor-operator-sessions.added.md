Added a YOURS / TEAM split to the local mirrord UI's session monitor: YOURS lists local `mirrord exec` sessions on this machine plus operator sessions whose owner matches the kubectl identity, TEAM lists everyone else's. The new operator-session detail surfaces locked ports, queue-splitting counts (SQS, RabbitMQ, Kafka), session uptime, and HTTP filter, and the same one-click Join row is available on your own local sessions, not just teammates'. The empty state ships a 3-step Connect operator wizard.
